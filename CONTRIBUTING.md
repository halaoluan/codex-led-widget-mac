# Contributing

Thanks for helping improve Codex LED Widget for macOS.

## Useful feedback

- macOS desktop behavior bugs.
- Quota reading issues.
- Plan label mismatches.
- UI layout problems on different screen sizes.
- Installation problems.

## Development

```bash
npm install
npm start
```

Build a release artifact:

```bash
npm run build
```

## Privacy

Please do not paste tokens, account secrets, or private Codex credentials in issues or pull requests.
